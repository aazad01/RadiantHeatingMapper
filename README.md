# Concept
Taking the dimensions of the room with some spacing off the wall.  Create a drawing of the PEX tubing that will circle back to the orignal position.

# Milestones
1. Using a square shape, 8 x 8 grid circle back with spacing in between.
2. Ability add zones
3. Export Measurements in SAE and Metric Units as PDF
3. Ability expand room square. i.e make room look like a Big square and small square on one side
4. Ability to add walls and no go zones 
5. Update zones or suggest zones.

# Inputs
1. Room # : Length, Width
2. Spacing for PEX Pipe
3. Blocker #: Starting Position (L, W), Length and Direction
